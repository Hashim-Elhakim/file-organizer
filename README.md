# File Organizer (Python Automation Tool)

A simple Python script that automatically organizes files into folders based on file type. Useful for cleaning up messy directories like Downloads.

## Features
- Automatically sorts files into categories
- Creates folders if they don't exist
- Handles unknown file types
- Beginner-friendly automation script

## How to Use
1. Open `organizer.py`
2. Set the folder path:
   TARGET_FOLDER = "C:/Users/YourName/Downloads"
3. Run the script:
   python organizer.py

## Requirements
- Python 3.8+
